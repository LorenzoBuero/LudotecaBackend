Ludoteca

Grupo 7 (Lorenzo Buero, Daniela Rodriguez, Lucas Medina y Bruno Kuznicki)

Ludoteca es el proyecto realizado por el grupo 7 para el curso de desarrollo web Codo a Codo 4.0.
Este sitio es un sitio web de reseñas y periodismo de videojuegos, contiene una página principal, una página para buscar reseñas, una página para cada reseña, una página que habla sobre nosotros y sobre el proyecto (esta página) y una página para contactarnos.
